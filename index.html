Investiga en diferentes sitios web los siguientes tópicos y describe cada uno de ellos:

Tema 1: Elementos de Páginas Web

I. Orígenes de la Web

  1. Web 2.0: Evolución de la web hacia un enfoque colaborativo donde los usuarios crean y comparten contenido.
  2. Web 3.0: También conocida como web semántica, busca que la información sea comprensible para las máquinas, utilizando inteligencia artificial.
  3. La W3C y los estándares: Organización que desarrolla estándares web para asegurar la calidad y compatibilidad de las tecnologías.

II. Lenguajes de etiquetas

  1. SGML: Un estándar que permite definir lenguajes de marcado.
  2. XML: Un lenguaje de marcado que permite definir estructuras de datos de manera flexible.
  3. HTML: El lenguaje básico para crear páginas web.
  4. XHTML: Una versión más estricta de HTML combinando la sintaxis de XML.
  5. HTML5: La última versión de HTML, con soporte para multimedia y nuevas funcionalidades.

III. Otros lenguajes relacionados al desarrollo web

  1. CSS/CSS3: Utilizado para definir la presentación de las páginas web.
  2. JavaScript: Lenguaje que añade interactividad a las páginas web.
  3. Ajax: Técnica que permite actualizar partes de una página sin recargarla completamente.
  4. Tecnologías del lado del servidor: Incluyen lenguajes como PHP, Python y Ruby para procesar datos en el servidor.
  5. Navegadores: Software que permite acceder a la web e interpretar el contenido.
  6. Motores renderizados: Componentes que procesan el código HTML y CSS para mostrar las páginas web.

IV. Herramientas de desarrollo

  1. Software WYSIWYG: Herramientas que permiten crear páginas web visualmente, como Adobe Dreamweaver.
  2. Otras alternativas:
    • Figma: Herramienta de diseño colaborativo para prototipos.
    • InVision: Plataforma para crear prototipos y colaboración en diseño.
    • Zeplin: Facilita la colaboración entre diseñadores y desarrolladores.
    • GitHub: Plataforma para el control de versiones y colaboración en proyectos.
  3. Inspección de código desde el navegador: Herramienta que permite ver y modificar el código fuente de las páginas web en tiempo real.

    4. II. Puntos básicos en la elaboración de una página Web

    Al crear una página web, es importante considerar:

    • La usabilidad y experiencia del usuario.
    • El diseño responsivo para dispositivos móviles.
    • La optimización para motores de búsqueda (SEO).
    • La velocidad de carga de la página.

    Tema 3: Estructura De Una Página Web

    I. Jerarquía de directorio de un proyecto web

    Una jerarquía de directorio típica incluye:

    • /index.html: Página principal del sitio.
    • /css: Carpeta que contiene archivos CSS.
    • /js: Carpeta que contiene archivos JavaScript.
    • /images: Carpeta que contiene imágenes utilizadas en la web.

    II. Elementos de estructura para el diseño de una página Web: Concepto

    1. Regla utilizada para validar y certificar una página web: Se deben seguir los estándares del W3C para asegurar la accesibilidad y compatibilidad.
    2. Reglas para escribir HTML: Mantener la semántica, usar correctamente las etiquetas y cerrar las etiquetas apropiadamente.
    3. Estructura básica de un archivo web: 
        <!DOCTYPE html>
        <html lang="es">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Título de la Página</title>
            </head>
            <body>
                <h1>Contenido Principal</h1>
                <p>Texto de ejemplo.</p>
            </body>
        </html>
